### Week 2 Outcomes:

* Grasped the AWS VPC architecture, including:
  * Subnets (Public / Private)
  * Route Tables
  * Internet Gateway / NAT Gateway
  * DHCP, VPC CIDR

* Understood the security components within VPC:
  * Security Groups (stateful)
  * Network ACLs (stateless)
  * Layered security model in a system.

* Understood VPC connectivity methods:
  * Site-to-Site VPN
  * Client VPN
  * Pros and cons of each type.

* Understood AWS Direct Connect:
  * Dedicated physical connection from on-premises to AWS
  * Used for systems requiring high bandwidth and strong security.

* Grasped Load Balancer knowledge:
  * Application Load Balancer (Layer 7)
  * Network Load Balancer (Layer 4)
  * High Availability & Fault Tolerance architecture.

* Capable of analyzing AWS network models and proposing appropriate architectures for different use cases.
